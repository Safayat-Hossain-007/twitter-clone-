### Setup .env file

```js
MONGO_URI=...
PORT=...
JWT_SECRET=...
NODE_ENV=...
CLOUDINARY_CLOUD_NAME=...
CLOUDINARY_API_KEY=...

```

### Build the FrontEnd(frontend will start from the frontend Directory
and the command is below)

```shell
cd frontend
npm install
npm run dev
```

### Start server (backend will start from the root directory)

```shell

open another terminal
npm install
npm start
```
